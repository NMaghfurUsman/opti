## controls

|       control      |           function            |
|-------------------:|:------------------------------|
|         `b`        |         toggle border         |
|         `a`        |      toggle always on top     |
|       `right`      |           next image          |
|       `left`       |         previous image        |
|       `drag`       |          move window          |
|     `ctrl+drag`    |           pan image           |
|    `drag edges`    | blurry scale image and window |
|`scroll` \| `up/down`| pixel scale image and window |
|`ctrl+scroll` \| `ctrl+up/down`| pixel scale image  |
